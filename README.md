# Python Web Scraper

## 📌 Overview

A Python web scraper built using **Requests** and **BeautifulSoup** to extract data from websites and save it in a structured format.

## ✨ Features

- Extracts website data
- Parses HTML pages
- Saves data to CSV
- Clean and readable code
- Easy to customize

## 🛠 Technologies

- Python
- Requests
- BeautifulSoup4

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Usage

```bash
python scraper.py
```
